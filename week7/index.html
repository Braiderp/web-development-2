Promises are a way to wait for code to complete before moving on.

You can use Async/Await to stop a promise until it completes

for example const result =await database.query(sqlStatement) the result variable contains the query result and would stop code until the query completed. Now you can safely use the result object

Ajax stands for Asynchronous Javascript and XML

You can make calls to using the fetch api like so: const url = 'https:example.com/data'; fetch(url) .then((response) => { if(response.ok) { return response; } throw Error(response.statusText); }) .then( response => // do something with response ) .catch( error => console.log('There was an error!') )
