Notes

  • Es6 supports object destructuring
  • Object destructuring is important because its used heavily in popular frameworks such as react
  • Object destructuring allows you to write much cleaner code.
  • There is a Document Object Model or Dom
  • Many frameworks such as react and vuejs use whats called a virtual dom witch is faster in general

Object Destructuring example

const obj = {name: "Braiden", age: 24}; const {name, age} = obj; //name = Braiden; // age = 24;

But i can also do this in a function like so

function display({name, age}){ return ` hello, ${name} you are ${age}` } display(obj);

I can do this with arrays

const array = ["bob", "21", "single"]; const [name, age, status] = array; // name = bob as its the first element. age = 21 as it is the second and status = single
