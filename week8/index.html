Notes

Canvas is a way to "draw" content on to an html element.

You should never style the width and height of a canvas with pure css "because the width and height attributes determine how large the canvas’s coordinate system is. If we don’t specify width and height , the canvas element will default to a width of 300 and a height of 150. If we set the width and height for a canvas only in CSS, the canvas element will be 300 by 150, and the CSS properties will simply determine how large the box is that displays the image."

You can do 3d renderings with webgl

You can draw images to them

You can take a HD image using canvas.toDataURL()

You can draw a circle on a canvas like so: function drawCircle(canvas) { var canvas = document.getElementById("myCanvas"); var context = canvas.getContext("2d"); context.beginPath(); context.arc(50, 50, 30, 0, Math.PI*2, true); }
